The pre-defined field types are the ones that are included in the base installation of Form Tools: no modules, just the Core script. If you chose to download one of the pre-packaged installations then it will include the File Upload and TinyMCE (WYSIWYG) rich-text editor field type modules as well. But let's start with the pre-defined field types. We'll look at them one by one, see what options are available for each, how they can be configured and a few other tips and tricks!
