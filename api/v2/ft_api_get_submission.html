The ft_api_get_submission() function does pretty much what you'd expect it to do: it retrieves all information about a form submission. Unlike many other API functions, this one is more for programmers. It returns a hash of information (where the keys are the database column names and the values are the actual values in the field, directly from the database) which you can use for whatever you need.
