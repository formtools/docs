{% include screenshot.html item="install_step1.png" %} Step 1 of the installation script is a welcome screen, and lets you choose your default language. Form Tools 3 comes with all languages as part of the installation, so you can choose any one you want from this screen. That language will be used for the installation script, and as the default language for your Form Tools installation.

Note: not all translations are complete, so if you fancy helping to translate, let us know.
