Settings

{% include screenshot.html item="i185.gif" %}

The Settings page contains a couple of settings for the module. Specificaly, the Export Manager allows you to generate actual files on the server instead of popping them open in a popup or prompting download through a dialog window. For the file generation option, here you can control which folder should be used to create and store the files.
