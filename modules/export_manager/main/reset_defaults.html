Reset Defaults

{% include screenshot.html item="i184.gif" %}

This page was added in the 2.0.0 version of the Export Manager. It lets you reset all the Export Manager to the "factory defaults". The problem was Form Tools 2.1 changed the way the submission data is stored in the database, which was central to the way the Export Manager worked. So when upgrading to 2.x of the Export Manager, you'll need to come here and reset your settings to the default. Be sure to read the page on Upgrading to 2.x: special instructions first, though!

This page can be handy any time you accidentally mess up your Export Group or Types.
