Chart Types

Line Charts Line Charts are just that! A single, coloured line showing the activity for each day / month. If you mouseover any of the points a tooltip will appear describing the day / month. {% include screenshot.html item="i301.gif" %}
Area Charts Area Charts are visually very much like Line Charts, except the area beneath the line is filled in. {% include screenshot.html item="i278.gif" %}
Column Charts Column Charts show the activity of each day/month in columns. As with the other chart types, you can mouseover the columns to see the details for that column. {% include screenshot.html item="i279.gif" %}
