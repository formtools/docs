Field Charts

{% include screenshot.html item="i282.gif" %}

The second type of visualization are called Field Charts: they display the breakdown of values in a single form field, showing the total count of each option. As such, you'll mostly want to use this visualization type with fields that have a finite list of input options - e.g. dropdowns, radio button and checkbox groups - anything that uses an Option List. You can use this with any field type, but you may get some pretty weird charts if there are hundreds of different values! (See the section on Tips for creating great charts).

Field Charts can displayed as Pie Charts, Column Charts and Bar Charts.
