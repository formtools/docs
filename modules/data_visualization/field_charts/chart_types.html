Chart Types

Pie Charts Pie Charts are a common way to visualize your field value breakdown. You can choose to display the pie chart as 2D or 3D. {% include screenshot.html item="i296.gif" %}
Bar Charts Bar Charts are like Column Charts, except they draw the bars horizontally instead of vertically. {% include screenshot.html item="i282.gif" %}
Column Charts Column Charts are also used to display Activity Charts. As with the other chart types, you can mouseover the columns to see the details for that column. {% include screenshot.html item="i279.gif" %}
