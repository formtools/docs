Client account expiry dates

Here's another example usage of the Hooks Manager module. This example works in conjunction with the Extended Client Fields module to add an expiry date option to client accounts.

For simplicity, we've split this example into two discrete steps: adding an expiry date field through the Extended Client Fields module, then adding a new rule through the Hooks Manager to check that client accounts are allowed to log in.
