Settings

{% include screenshot.html item="i225.jpg" %}

The Settings page was added in version 1.1.0 of this module (July 2011). It contains a single setting: it lets you choose to display a "Back up form" button at the bottom of the administrator's main Forms page so you can link directly to the module. See the screenshots on this page to see and how the button looks.

If backing-up/copying your forms is something you do quite often, this can save you a few annoying clicks to get to the page you want!
