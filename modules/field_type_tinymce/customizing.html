Customizing individual fields

What if you need different fields to have different configurations? No problem. All you need to do is override the default values for that field. In the Edit Form -> Fields tab, click the edit icon. That will open up a dialog window with two tabs: Main Settings & WYSIWYG settings. There, you can override any settings by unchecking the “Default Value” checkbox and selecting the value you want.
