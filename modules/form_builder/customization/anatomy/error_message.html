Error Message

The Error Message template is used to display any errors that occur when a form submission fails server-side validation. For the majority of users, if a field contains field validation and they fail to enter the value properly, a dialog window will appear containing the error message(s). However, for people with browsers that have javascript disabled, the errors will be caught on the server (* N.B. At the current version, not all client-side code has corresponding server-side validation. Read the section on Field Validation for more information.
