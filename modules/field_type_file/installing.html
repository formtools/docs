Installing

In order to use the file upload field type in Form Tools 2.1.x, you'll need to first install the module. Download the latest version from the Modules section above. Then, follow the instructions on this page. That explains how to install any Form Tools module.

Once the module has been installed, on the Edit Form -> Fields tab you'll see a new "File" field type in the Field Types dropdowns for each field.

The next page explains the various options for your form fields.
